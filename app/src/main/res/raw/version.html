Version 2.1

  • Fixed some problems if a book is missing some tracks.
  • Lowered the memory usage for book covers. Now a scaled version of the bitmap is used.

Version 2.0

  • Dropbox integration for automatic syncing with this application's dropbox folder.
  • Background library scanning so large libraries can be scanned while the application is still fully usable.
  • Support for the media button on some headphones for easy play/pause functionality.
  • Added some one time display tutorial screens.
  • Fixed a multitude of small bugs.

Version 1.0

  • Shake the device every once and a while to keep the audiobook playing, otherwise the book will slowly mute and pause. Easy to turn on and off with the push of a button.
  • Books will always remember what position they are at so you can continue listening to it from the some spot in the future.
  • Book is paused if you unplug your headphones.
